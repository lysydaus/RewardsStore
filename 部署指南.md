# 微信分享部署指南

## 最简单的方法：使用 Vercel（推荐）

### 步骤1：准备工作
1. 访问 https://vercel.com/
2. 使用 GitHub、GitLab 或邮箱注册账号（免费）

### 步骤2：部署项目

#### 方法A：通过网页上传（最简单）
1. 登录 Vercel
2. 点击 "Add New..." → "Project"
3. 选择 "Import Git Repository" 或直接拖拽 RewardsStore 文件夹
4. 点击 "Deploy"
5. 等待几分钟，部署完成

#### 方法B：使用命令行
1. 安装 Node.js（如果还没有）
2. 打开终端，安装 Vercel CLI：
   ```bash
   npm install -g vercel
   ```
3. 进入项目目录：
   ```bash
   cd /path/to/RewardsStore
   ```
4. 登录并部署：
   ```bash
   vercel login
   vercel
   ```
5. 按提示操作，几分钟后获得链接

### 步骤3：获取链接
部署成功后，你会得到一个链接，例如：
```
https://rewards-store-xxx.vercel.app
```

### 步骤4：在微信中分享
1. 复制链接
2. 在微信中发送给好友或群
3. 点击链接即可在微信内置浏览器中使用

## 方法2：使用 Netlify

### 步骤1：注册
1. 访问 https://www.netlify.com/
2. 注册免费账号

### 步骤2：部署
1. 登录后，点击 "Add new site" → "Deploy manually"
2. 将整个 RewardsStore 文件夹拖拽到页面中
3. 等待上传和部署完成

### 步骤3：获取链接
部署成功后，你会得到一个链接，例如：
```
https://rewards-store-xxx.netlify.app
```

---

## 方法3：使用 GitHub Pages

### 步骤1：上传到 GitHub
1. 在 GitHub 创建新仓库
2. 上传 RewardsStore 文件夹中的所有文件

### 步骤2：启用 GitHub Pages
1. 进入仓库设置（Settings）
2. 找到 "Pages" 选项
3. 在 "Source" 中选择 "main" 分支
4. 点击 "Save"

### 步骤3：访问链接
几分钟后，你的网站会发布到：
```
https://你的用户名.github.io/仓库名/
```

## 生成二维码

部署完成后，可以生成二维码方便分享：

1. 访问二维码生成网站：
   - https://cli.im/（草料二维码）

3. 生成并下载二维码图片

4. 在微信中分享二维码图片

## 自定义域名（可选）

如果你有自己的域名，可以绑定到部署的网站：

### Vercel
1. 在项目设置中找到 "Domains"
2. 添加你的域名
3. 按提示配置 DNS

### Netlify
1. 在站点设置中找到 "Domain management"
2. 添加自定义域名
3. 按提示配置 DNS

## 常见问题

### Q: 部署后数据会丢失吗？
A: 不会。每个用户的数据存储在自己浏览器的 localStorage 中。

### Q: 可以多人同时使用吗？
A: 可以。每个用户都有独立的数据。

### Q: 如何更新网站？
A: 
- Vercel/Netlify：重新上传文件或推送到 Git 仓库
- GitHub Pages：更新 GitHub 仓库中的文件

### Q: 微信中打不开怎么办？
A: 
1. 确保使用的是 https 链接
2. 检查链接是否正确
3. 尝试在微信中点击右上角"..."，选择"在浏览器中打开"

### Q: 如何对接真实的后端？
A: 修改 `db.js` 文件中的配置：
```javascript
const DB_ADAPTER = 'api';
const API_BASE_URL = 'https://your-api.com';
```

3. 尝试清除浏览器缓存后重新访问

# 振华商城 - 会员积分系统
 
一个完整的HTML5商城系统，包含会员管理、积分系统、站内消息通知等功能。

## 🚀 快速开始

### 本地运行

1. 克隆项目到本地
2. 使用浏览器打开 `index.html`
3. 选择入口：
   - **顾客入口**：购物、会员开通
   - **商家入口**：订单管理、商品管理
   - **运维人员**：系统配置、用户管理

### 在线部署

项目可以直接部署到GitHub Pages、Netlify、Vercel等静态网站托管平台。

**注意**：首次访问时，系统会自动初始化演示数据。

## 🔔 会员通知系统

### 自动提醒功能

系统会在准会员到期前自动发送提醒：

- **T-3天**（还剩3天）：提醒已节省费用，建议开通正式会员
- **T-1天**（还剩1天）：提醒明天到期，限时优惠¥169
- **T-0天**（当天到期）：今天到期，限时优惠即将失效

### 走马灯展示

所有页面顶部会显示最新的会员提醒消息，支持：
- 从右向左滚动
- 鼠标悬停暂停
- 一键开通会员
- 查看详情
- 关闭提醒

### 站内短信

在"我的"页面可以查看所有站内消息：
- 消息列表
- 已读/未读状态
- 未读消息统计
- 一键全部已读

## 🎯 特色功能

### 用户006自动递减

用户006（13800000006）的到期时间会每天自动递减：
- 第1天：T+3（还剩3天）
- 第2天：T+2（还剩2天）
- 第3天：T+1（还剩1天）
- 第4天：T+3（循环回3天）

这个功能方便测试不同时间点的提醒效果。

### 限时优惠

准会员用户可享受一次限时优惠：
- 原价：¥199/年
- 优惠价：¥169/年（85折）
- 每个用户仅可享受1次
- 在会员开通页面自动显示

## 🛠️ 技术栈

- **前端框架**：原生HTML5 + Tailwind CSS

- **数据存储**：localStorage（可切换到后端API）
- **语言切换**：中文/英文

## 📦 数据存储

系统使用localStorage存储数据，优点：
- ✅ 无需服务器
- ✅ 可直接打开HTML文件运行
- ✅ 数据持久化

缺点：
- ⚠️ 数据存储在浏览器本地
- ⚠️ 清除浏览器数据会丢失
- ⚠️ 不同浏览器数据不共享

### 切换到后端API

在 `db.js` 中修改配置：

```javascript
const DB_ADAPTER = 'api'; // 从 'local' 改为 'api'
const API_BASE_URL = 'https://your-api.com'; // 设置API地址
```

## 🔧 重置数据

如果需要重置所有数据，在浏览器控制台执行：

```javascript
DB.reset()
```

这会清空所有数据并重新初始化演示数据。
